    DeleteChannel

    This action deletes the channel from memory and the database.

    Parameters

    • channel(string): Name of the channel to be removed.

    • hash(string): password of the channel to delete.

    Example

    {
    "DeleteChannel": {
        "channel": "Fire",
        "hash": "739"
    }
}
